### Project Description
An enhanced Profile Management System built using the Java Web MVC architecture (Servlet, JSP, JavaBean, and JDBC) connected to a MySql database.

### Implemented Features
1. **Insert Profile:** Register and save student details into the database.
2. **View Profiles:** Display all registered profiles from the database in a neat table format.
3. **Delete Profile (Additional Feature - Option C):** Users can safely remove a student's profile record from the system.
